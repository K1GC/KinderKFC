# KinderKFC - Ticket and Product`s Discord bot

![KinderKFC Logo](https://i.postimg.cc/MKF712hZ/kinderkfclogo2.png)

A powerful Discord bot designed for managing in-game item shops with ticket-based ordering system.

## Features

- **Product Management** - Add, edit, remove products with custom prices and images
- **Shopping Cart** - Users can add/remove items, view their basket
- **Ticket System** - Automatic order tickets with admin notifications
- **Welcome/Leave Messages** - Customizable join/leave messages with images
- **Customizable UI** - Colors, titles, descriptions for buy prompts
- **Role Management** - Admin role and allowed roles for bot usage
- **Logging** - Order creation and closure logs
- **Minimum Order Amount** - Set minimum order value

## Requirements

- Node.js v18.0.0 or higher
- Discord Bot Token
- Server with admin permissions
